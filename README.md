# Pyhton-Password_Strength_Checker
A simple Python script that evaluates password strength based on length, numbers, uppercase letters, and special characters.

This project is a beginner-friendly Password Strength Checker written in Python.
It evaluates a user’s password and classifies it as Weak, Moderate, or Strong by checking for:

Minimum length (8 characters or more)

Presence of uppercase and lowercase letters

Inclusion of digits and special characters

The project demonstrates basic Python concepts such as string manipulation, loops, conditional statements, and user input/output. It’s an excellent starter project for anyone learning Python and cybersecurity fundamentals related to password hygiene.
